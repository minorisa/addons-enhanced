List View Limit Options

This module enables you to set different options in the records per page drop-down of List views. Options are defined in Settings Configuration Website Settings List View Limit Options.

Available options are:

  • Limit Stops: Comma-separated list of integers that will be shown as options in the drop-down of records per page of the List View. Default value of 80 records per page will be always included (no need to specify it) as first option. On change method of this field will try to clean invalid data.
  • Hide Unlimited Option: If checked, the Unlimited option will be hidden and thus unselectable.
